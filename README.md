# E-Commerce Sales Analysis Dashboard

## Project Overview
Analyzed an e-commerce dataset to identify sales trends, top products, and regional performance.

## Tools Used
- Python (Pandas, NumPy)
- Power BI
- Excel

## Key Insights
- Sets generate nearly 50% of total revenue
- Maharashtra contributes the highest sales
- Medium and Large sizes dominate demand
